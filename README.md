## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Set environment variables
```bash
  export NEXT_PUBLIC_BACKEND_URL=<your_locally_running_backend_url>
```

Start the server

```bash
  yarn dev
```
